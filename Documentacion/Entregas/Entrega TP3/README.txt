Entregable del analizador Semantico


Descripcion del mapa del proyecto:


- Codigo Fuente: Se encuentra el codigo fuente (.java) del Analizador Semantico

- Docs: Javadocs del codigo fuente

- Documentacion: Se encuentra el .doc descriptivo del proyecto y documentos adicionales utilizados

- Ejecutables: Se encuentran el ejecutable (.jar). Para ejecutarlos se debe poner en la consola
java -jar <Version a ejecutar> "Path al archivo a compilar"

- Proyectos de Eclipse: Se encuentran el proyecto de Eclipse para poder cargarlo sin necesidad alguna

- Test: Se encuentran las pruebas realizadas



ACLARACION: Puede que en Linux tenga problemas con el direccionamiento a los archivos.
