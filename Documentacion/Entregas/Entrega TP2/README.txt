Este entregable consta de dos versiones del Analizador Sintactico como se explica en el .doc
La unica diferencia entre la version 1 y la 2 es que la 1 implementa un manejo estricto de errores, ante el primer error
sintactico, se corta la ejecucion del anaisis sintactico terminando el analisis lexico. En la version 2 en cambio se emplea
una especie de sincronizacion por bloques, es decir, que el analisis continua.


Descripcion del mapa del proyecto:


- Codigo Fuente: Se encuentra el codigo fuente (.java) de ambas versiones del analizador sintactico (1 y 2)

- Docs: Javadocs del codigo fuente de ambas versiones

- Documentacion: Se encuentra el .doc descriptivo del proyecto y documentos adicionales utilizados

- Ejecutables: Se encuentran los ejecutables (.jar) correspondiente a ambas versiones. Para ejecutarlos se debe poner en la consola
java -jar <Version a ejecutar> "Path al archivo a compilar"

- Proyectos de Eclipse: Se encuentran los proyectos de Eclipse para poder cargarlos sin necesidad alguna

- Test: Se encuentran las pruebas realizadas
